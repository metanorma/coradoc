== Code Examples

Literal block without syntax highlighting:

----
def hello_world
  puts "Hello, World!"
end
----

Listing block with syntax highlighting:

[source,ruby]
----
class Greeter
  def initialize(name)
    @name = name
  end

  def greet
    puts "Hello, #{@name}!"
  end
end
----
