= Ruby Methods Reference

This reference documents all Ruby methods available in Coradoc's public API.

== Main Module Methods

=== Coradoc.parse

Parse a document string into a model.

.Syntax
[source,ruby]
----
Coradoc.parse(input, format: :asciidoc, **options) -> CoreModel::Base
----

.Parameters
[horizontal]
`input`:: `String` - Document content
`format`:: `Symbol` - Input format (:asciidoc, :markdown, :html)
`schema`:: `Symbol` - Validation schema (:default, :iso)
`strict`:: `Boolean` - Raise on parse errors (default: false)
`attributes`:: `Hash` - Document attributes

.Returns
`Coradoc::CoreModel::StructuralElement` (document)

.Examples
[source,ruby]
----
# Parse AsciiDoc
doc = Coradoc.parse("= Title\n\nContent", format: :asciidoc)

# Parse Markdown
doc = Coradoc.parse("# Title\n\nContent", format: :markdown)

# Parse HTML
doc = Coradoc.parse("<h1>Title</h1><p>Content</p>", format: :html)

# With strict mode
begin
  doc = Coradoc.parse(input, format: :asciidoc, strict: true)
rescue Coradoc::ParseError => e
  puts "Error at line #{e.line}: #{e.message}"
end
----

=== Coradoc.convert

Convert a document between formats.

.Syntax
[source,ruby]
----
Coradoc.convert(input, from:, to:, **options) -> String
----

.Parameters
[horizontal]
`input`:: `String` - Document content
`from`:: `Symbol` - Source format (:asciidoc, :markdown, :html)
`to`:: `Symbol` - Target format (:asciidoc, :markdown, :html)
`theme`:: `Symbol` - HTML theme (:classic, :modern, :minimal)
`validate`:: `Boolean` - Validate before conversion

.Returns
`String` - Converted document

.Examples
[source,ruby]
----
# Markdown to HTML
html = Coradoc.convert("# Title", from: :markdown, to: :html)

# AsciiDoc to Markdown
md = Coradoc.convert("= Title\n\nContent", from: :asciidoc, to: :markdown)

# HTML to AsciiDoc
adoc = Coradoc.convert("<h1>Title</h1>", from: :html, to: :asciidoc)

# With theme
html = Coradoc.convert(adoc, from: :asciidoc, to: :html, theme: :modern)
----

=== Coradoc.serialize

Serialize a CoreModel to a format string.

.Syntax
[source,ruby]
----
Coradoc.serialize(model, to:, **options) -> String
----

.Parameters
[horizontal]
`model`:: `CoreModel::Base` - Document model
`to`:: `Symbol` - Target format
`indent`:: `Integer` - Output indentation
`line_width`:: `Integer` - Maximum line width

.Returns
`String` - Serialized document

.Examples
[source,ruby]
----
# Parse then serialize
doc = Coradoc.parse(input, format: :asciidoc)
html = Coradoc.serialize(doc, to: :html)

# With formatting options
adoc = Coradoc.serialize(doc, to: :asciidoc, indent: 2, line_width: 80)
----

== CoreModel Methods

=== StructuralElement

[source,ruby]
----
# Create a document
doc = Coradoc::CoreModel::StructuralElement.new(
  element_type: "document",
  title: "My Document",
  id: "my-doc",
  children: []
)

# Create a section
section = Coradoc::CoreModel::StructuralElement.new(
  element_type: "section",
  level: 1,
  title: "Introduction",
  children: []

# Access methods
doc.element_type  # => "document"
doc.title         # => "My Document"
doc.id            # => "my-doc"
doc.children      # => []

== Document Manipulator

=== DocumentManipulator

[source,ruby]
----
require 'coradoc/document_manipulator'

# Create manipulator
manipulator = Coradoc::DocumentManipulator.new(doc)

# Chain operations
result = manipulator
  .transform_text(&:upcase)
  .transform_headings(&:upcase)
  .add_toc(levels: 3, position: :top)
  .add_metadata("processed_at" => Time.now.iso8601)
  .remove_elements(:comment_line)
  .to_html

# Available methods
manipulator.transform_text(&:method)     # Transform all text
manipulator.transform_headings(&:method) # Transform headings
manipulator.add_toc(options)             # Add table of contents
manipulator.add_metadata(hash)           # Add metadata
manipulator.remove_elements(type)        # Remove element types
manipulator.to_html                      # Output as HTML
manipulator.to_markdown                  # Output as Markdown
manipulator.to_asciidoc                  # Output as AsciiDoc
----

== Query API

=== Query Methods

[source,ruby]
----
# Parse document
doc = Coradoc.parse(input, format: :asciidoc)

# Find by element type
sections = doc.query('section')
paragraphs = doc.query('paragraph')
tables = doc.query('table')

# Find by ID
intro = doc.query('#intro').first

# Find by class/role
examples = doc.query('.example')

# Complex selectors
results = doc.query('section > paragraph:first-child')

# Query within element
section = sections.first
paras = doc.query_within(section, 'paragraph')

# Chain queries
doc.query('section')
   .filter { |s| s.level == 1 }
   .map { |s| s.title }
----

=== Selector Syntax

[horizontal]
`element`:: Element type (section, paragraph, table)
`#id`:: ID selector
`.class`:: Class/role selector
`[attr=value]`:: Attribute selector
`:first-child`:: Pseudo-class selectors
`>`:: Child combinator
(space):: Descendant combinator

== Streaming API

=== Streaming Methods

[source,ruby]
----
require 'coradoc/streaming'

# Stream parse large file
Coradoc::Streaming.parse_large_file("large.adoc", format: :asciidoc,
                                    chunk_size: 100) do |chunk|
  chunk.each { |element| process_element(element) }
end

# Transform in chunks
results = Coradoc::Streaming.transform_in_chunks(elements, chunk_size: 50) do |chunk|
  chunk.map { |el| transform_element(el) }
end

# Incremental serialization
File.open("output.html", "w") do |file|
  Coradoc::Streaming.serialize_incremental(document, format: :html) do |chunk|
    file.write(chunk)
  end
end

# Memory-limited processing
progress = Coradoc::Streaming.process_with_memory_limit(
  "input.adoc", "output.html",
  format: :asciidoc, output_format: :html,
  max_memory: 50 * 1024 * 1024  # 50MB
)
puts progress.to_s  # "100 processed (100.0%) at 10.0/sec"
----

== Validation API

=== Validation Methods

[source,ruby]
----
require 'coradoc/validation'

# Define a schema
schema = Coradoc::Validation::Schema.define do
  required :title, type: String, min_length: 1
  required :sections, type: Array, min_count: 1
  optional :author, type: String

  rule :check_references do |doc|
    refs = doc.query('xref')
    missing = refs.reject { |r| doc.resolve_reference(r) }
    missing.map { |r| "Unresolved reference: #{r.target}" }
  end
end

# Validate document
result = schema.validate(document)

if result.valid?
  puts "Document is valid"
else
  result.errors.each { |e| puts "#{e.path}: #{e.message}" }
end
----

=== Built-in Validation Rules

[horizontal]
`required`:: Field must be present
`type`:: Field must be specific type
`min_length`/`max_length`:: String/collection length bounds
`min_count`/`max_count`:: Collection count bounds
`format`:: Match against regex pattern
`rule`:: Custom validation block

== Lazy Evaluation API

=== Lazy Methods

[source,ruby]
----
require 'coradoc/lazy'

# Wrap document for lazy iteration
wrapper = Coradoc::Lazy.wrap(document)
wrapper.each_section do |section|
  process_section(section)  # Processed on-demand
end

# Lazy transformation pipeline
result = Coradoc::Lazy.transform(sections) do |p|
  p.map { |s| transform_section(s) }
   .select { |s| s.visible? }
   .take(10)
end.to_a  # Only evaluates when to_a is called

# Process in batches
wrapper.each_batch(10) do |batch|
  batch.each { |section| process(section) }
end

# Lazy reference resolution
resolver = Coradoc::Lazy.resolver(document, loader: ->(ref, _) {
  load_include_file(ref)
})
content = resolver.resolve("include::chapter1.adoc[]")
----

== Error Handling

=== Error Classes

[source,ruby]
----
# Error hierarchy
Coradoc::Error                          # Base error class
├── Coradoc::ParseError                 # Parsing errors
├── Coradoc::ValidationError            # Validation errors
├── Coradoc::TransformationError        # Transform errors
├── Coradoc::UnsupportedFormatError     # Unknown format
└── Coradoc::ConfigurationError         # Config errors

# Handling errors
begin
  doc = Coradoc.parse(input, format: :asciidoc, strict: true)
rescue Coradoc::ParseError => e
  puts "Parse error at line #{e.line}, column #{e.column}"
  puts e.message
rescue Coradoc::ValidationError => e
  puts "Validation failed: #{e.message}"
rescue Coradoc::UnsupportedFormatError => e
  puts "Unknown format: #{e.format}"
end
----

== Configuration

=== Global Configuration

----
# Configure globally
Coradoc.configure do |config|

  config.strict = false
  config.warn_on_undefined = true
  config.parser_options = { preserve_whitespace: true }
end

# Access configuration
Coradoc.config.default_format  # => :asciidoc

# Debug mode
Coradoc.debug = true
