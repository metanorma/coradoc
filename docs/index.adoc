== Purpose

Coradoc is a parser, model, and serializer for AsciiDoc documents.

It provides a complete toolchain for working with AsciiDoc:

* *Parsing* - Convert AsciiDoc text into a structured document model
* *Manipulation* - Modify document content through Ruby objects
* *Serialization* - Generate AsciiDoc output from document models
* *Round-tripping* - Parse and serialize while preserving semantic meaning

Coradoc uses https://parslet.rubygem[Parslet] for parsing and https://github.com/metanorma/lutaml-model[lutaml-model] for model definition, providing a robust foundation for AsciiDoc document processing.

== Quick Start

[source,ruby]
----
# Parse AsciiDoc input
doc = Coradoc.parse("== Hello World\n\nThis is a paragraph.")

# Access document structure
doc.header.title                    # => "Hello World"
doc.sections.first.title.level_int  # => 2

# Serialize back to AsciiDoc
output = doc.to_adoc
----

== Installation

Add this line to your application's Gemfile:

[source,ruby]
----
gem 'coradoc'
----

And then execute:

[source,bash]
----
bundle install
----

Or install it yourself as:

[source,bash]
----
gem install coradoc
----
