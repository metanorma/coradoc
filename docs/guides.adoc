= Guides

== Purpose

This section contains task-oriented guides to help you accomplish specific goals with Coradoc. Each guide focuses on a particular task or problem.

== Overview

Guides are organized by task:

* link:/coradoc/guides/ruby-api/[Ruby API Guide] - Complete Ruby API usage guide
* link:/coradoc/guides/custom-blocks/[Custom Blocks] - Create custom block types
* link:/coradoc/guides/custom-inline/[Custom Inline Elements] - Create custom inline elements
* link:/coradoc/guides/schema-system/[Schema System] - Use schemas for document types
* link:/coradoc/guides/configuration/[Configuration] - Configuration options reference
* link:/coradoc/guides/troubleshooting/[Troubleshooting] - Common issues and solutions

== Guide Structure

Each guide follows this structure:

* **Task** - What you want to accomplish
* **Approach** - Recommended solution
* **Steps** - Detailed implementation
* **Examples** - Working code samples
* **Caveats** - Things to watch out for

== When to Use Guides vs. Tutorials

* **Tutorials** - Learn step-by-step from basics
* **Guides** - Solve specific problems (assume some knowledge)
